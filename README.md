# Gloss-Mod-Manager
Gloss Mod Manager is a lightweight universal mod manager for PC games that installs, organizes, and toggles mods across Unreal Engine and REEngine titles. Supports Nier Automata, Resident Evil 4, Monster Hunter, Dragon's Dogma 2, Hogwarts Legacy, and Devil May Cry 5. Alternative to Fluffy Quack and Vortex.
